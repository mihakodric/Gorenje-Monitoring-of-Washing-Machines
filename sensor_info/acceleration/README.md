## LIS2DW12 Triple Axis Accelerometer (SKU SEN0405)

Website: [LIS2DW12 Triple Axis Accelerometer (DFRobot)](https://wiki.dfrobot.com/LIS2DW12_Triple_Axis_Accelerometer_SKU_SEN0405)

## Pin Connections (Sensor → ESP32)

- VCC → 3.3V
- GND → GND
- SCL → GPIO 22
- SDA → GPIO 21
